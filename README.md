# Aprendendo a Aprender

## Problema 📋

O objetivo deste projeto foi construir uma página web informativa sobre técnicas de aprendizado eficaz, baseada em conteúdo educacional e estruturada de forma clara e organizada.

O desafio principal foi consolidar fundamentos de HTML e CSS, aplicando semântica, organização com BEM e estrutura modular, mantendo consistência visual e legibilidade.

---

## Solução 🎯

Foi desenvolvida uma página estática que apresenta conceitos fundamentais sobre aprendizado eficiente, incluindo:

- Problemas do modelo tradicional de ensino
- Técnicas comprovadas de estudo
- Estratégias cognitivas (como pensamento difuso e focado)
- Métodos práticos como a Técnica de Feynman
- Dados relevantes sobre o cérebro e aprendizagem

A proposta é entregar conteúdo educativo de forma estruturada e acessível, com foco em leitura e absorção de conhecimento.

## Decisões Técnicas 🧠

- Uso de **HTML semântico** para melhor estruturação do conteúdo.
- Aplicação da metodologia **BEM Flat** para organização do CSS.
- Arquitetura modular com separação por blocos.
- Uso de:
  - `flexbox` para layout

- Inclusão de **normalize.css** para padronização entre navegadores.
- Implementação de **animações com CSS** para enriquecer a experiência visual.

### Limitação assumida

Este projeto **não possui responsividade completa**, decisão tomada para focar na base estrutural e organização do código.
